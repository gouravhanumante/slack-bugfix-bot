# Custom Rules

Place your team's coding rules and guidelines here as `.md` or `.txt` files.
All files in this directory are loaded into the AI agent's system prompt.

Example files:
- `coding-standards.md` — naming conventions, formatting rules
- `architecture.md` — project structure, design patterns
- `kmp-patterns.md` — KMP-specific patterns to follow

The AI will follow these rules when fixing bugs.
